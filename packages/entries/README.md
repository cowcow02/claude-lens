# @claude-lens/entries

Day-scoped Entry primitives for Fleetlens — one artifact per (session × local-day).

Builds on top of `@claude-lens/parser` session data to produce structured entries that represent meaningful work units: what happened, in which project, on which day.

## Exports

- `@claude-lens/entries` — types, signals, and entry builder (pure, browser-safe)
- `@claude-lens/entries/fs` — filesystem storage layer (node:fs only)
