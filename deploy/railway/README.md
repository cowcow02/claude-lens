# Deploy to Railway

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/new/template?template=https://github.com/cowcow02/fleetlens)

## Steps

1. Click the button above (or import this repo from the Railway dashboard).
2. Railway auto-provisions Postgres and generates `POSTGRES_PASSWORD`. Add `DATABASE_URL` pointing to the Postgres service and set `BASE_URL` to your public domain.
3. Deploy. The team server starts on port 3322 and is exposed via Railway's generated domain.
